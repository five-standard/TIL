# Protocol
  > **통신 규약**

  컴퓨터 사이에서 정보를 주고받을 때 사용하는 통신 규약이다.

  ## 종류
  |계층|프로토콜|
  |---|---|
  |응용|HTTP, SMTP, Telnet, etc...|
  |표현|ASCII, MPEG, JPEG, etc...|
  |세션|NetBios, SAP, SDP, etc...|
  |전송|TCP, UDP, SPX|
  |네트워크|IP, IPX|
  |데이터 링크|Ethernet, Token Ring, FDDI, etc..|
  |물리|-|

  ## 기능
  - ### 세분화와 재합성

  - ### 캡슐화

  - ### 연결제어

  - ### 오류제어

  - ### 흐름제어

  - ### 동기화

  - ### 순서결정

  - ### 주소설정

  - ### 다중화
