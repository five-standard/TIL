# TailwindCSS

> **유틸리티를 우선으로 한 스타일링 프레임워크**

유틸리티 클래스를 통해 손쉽게 스타일링을 할 수 있게 해주는 프레임워크이다.

## 장단점

### 장점

- **스타일 컴포넌트를 따로 만들 필요가 없다**
- **컴포넌트의 이름을 고민할 필요가 없다**
- **퍼블리싱을 빠르게 진행할 수 있다**

### 단점

- **러닝커브가 높은 편이다**
- **제대로 된 CSS 학습에 방해된다**
- **코드의 가독성이 낮아진다**

### 예시

```js
return <>
  <div className="flex items-center">
    <p className="mt-8 w-full bg-white rounded-8">
      hi
    </p>
  </div>
<>
```
