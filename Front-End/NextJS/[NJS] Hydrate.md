# Hydrate

> **HTML과 JS를 매칭시키는 과정**

서버에서 렌더링 된 정적 페이지와 JS 파일을 매칭시키는 과정이다.

## 렌더링 두번 에반데

렌더링을 서버와 클라이언트 모두에게서 진행하니까 비효율적이라고 생각할 수 있다.  
 하지만 이미 서버에서 렌더링 된 가벼운 파일을 빠르게 전송할 수 있다는 장점이 있다.
