매일매일 조금씩이라도 공부하기 위해 만든 레포지토리입니다.  

## 규칙
```
  # (제목)
  > ㅤ  
  > **(주제에 대한 간단한 설명)**
  > ㅤ

  (주제에 대한 구체적인 설명)

  ## 하위 내용 //세부 내용
    ### 세부 내용 1
      (설명 1)
    ### 세부 내용 2
      (설명 2)

  ## 하위 내용 2 //일반 내용 (주제에 대한 특징, 사용법 등을 작성)
    (설명)
  
  ## 하위 내용 3 //목록 (한 주제에 대한 내용이 여러개인 상황에서 사용)
    - (내용 1)
    - (내용 2)
    - (내용 3)
      - (내용 설명 1)
      - (내용 설명 2)
```
